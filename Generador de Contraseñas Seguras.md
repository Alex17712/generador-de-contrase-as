# Objetivo del Programa
Desarrollar un programa capaz de generar contraseñas seguras de manera automática, utilizando estructuras condicionales, repetitivas y funciones en Python.
Este programa genera contraseñas aleatorias conbinando
Letras Mayusculas
Letras Minusculas
Numeros
Caracteres Especiales
# Funcionalidades Principales
Solicita al usuario la longitud de la contraseña
Valida que el número ingresado sea mayor al mínimo permitido
Si el usuario ingresa un número incorrecto, lo vuelve a pedir
Genera contraseñas aleatorias seguras
Usa funciones para mejorar la organización del código
Aplica estructuras:
# Condicionales (if / else)
# Repetitivas (while)
# Uso de librerías para aleatoriedad
# Funcionamiento del Programa
El usuario ingresa la longitud de la contraseña.
El sistema valida que sea un número válido.
Si el número es menor al permitido:
Se muestra un mensaje de error.
Se vuelve a solicitar el dato.
Una vez validado:
Se genera automáticamente una contraseña segura.
# Tecnologías Utilizadas
Python
Librería random
Funciones definidas por el usuario
# Resultado Esperado
El programa permite generar contraseñas seguras que ayudan a proteger la información del usuario, evitando claves débiles o fáciles de adivinar.
